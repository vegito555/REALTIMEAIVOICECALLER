Credentials and infra config are loaded only from real environment variables set on your VPS / Coolify dashboard. They are never written to or read from the Supabase settings table.

The fields below are read-only diagnostics. Each shows whether the corresponding env var is configured. To change a value, update it in your hosting provider and redeploy. Only system_prompt and ENABLED_TOOLS are editable from this dashboard.

🔧 Quick-Start Guide

  1. Run supabase_schema.sql in Supabase SQL Editor.
  2. Open ⚙️ Settings → fill LiveKit, Gemini, and Vobiz credentials → Save.
  3. Click ⚡ Create SIP Trunk on the Vobiz card.
  4. Open ✏️ AI Prompt → tweak for your business → Save.
  5. (Optional) Open 🤖 Agents → create at least one named agent profile.
  6. Open 📞 Single Call → call your own number to test.
